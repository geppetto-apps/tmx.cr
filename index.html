Tmx.cr

Tmx loader for loading Tiled maps into a basic data structure in Crystal.

Installation

Add this to your application's shard.yml:

dependencies:
  tmx:
    github: geppetto-apps/tmx.cr

Usage

require "tmx"

map = Tmx.load(some_path)
map.layers
